Budget App - This project is a budget app that allows the user to input income and budget items. The app automatically allocates amounts for each item based on priority levels assigned by the user. This app is Task 1 for Team Hera working together from one repository for th HNG Internship 6.0.

Hera helps you to allocate finance(funds) using a priority algorithm, this algorithm uses priority assigned to items like savings, foods, and clothings to allocate funds based on your budget thereby resolving the task of having to worry about budgeting.
